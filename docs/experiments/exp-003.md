# Phase 0 full_auto 27機場

## 1. 目的(Why)

exp-002（quick モード）との比較のため、同じ27機場データで full_auto モードを実行し
RMSE の改善幅を確認する。

## 4. 考察(Discussion)

- RMSE 0.882 は quick（0.763）より悪化。full_auto で追加されたモデル群が
  Group CV スコアを押し下げた可能性がある。
- ベストモデルは exp-002 と同じ LightGBM on ElasticNet。full_auto でも
  このブループリントが上位に残ったことは再現性として良いシグナル。
- Group CV（プラグ単位）はデータ量が多い分、fold 間の分散が大きくなりやすく、
  RMSE の絶対値だけで quick/full_auto を比較するのは注意が必要。
- テスト予測 CSV が生成された（exp-002 は未生成）。予測分布の確認が可能。

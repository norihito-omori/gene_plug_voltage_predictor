# DataRobot TS t+1-t+7 27機場

## 1. 目的(Why)

Group CV（プラグ単位）による `future_7d_max` 予測（exp-001〜004）から DataRobot
Time Series（Datetime Partitioning + Multiseries）による `daily_max` の t+1〜t+7
予測へ移行し、時間的リークのない評価で実運用に近い精度を計測する。
7 点予測の max を閾値判定に使うことで `future_7d_max` と同等の運用判断が可能になる。

## 4. 考察(Discussion)

- RMSE 0.337 は exp-004（Group CV + trend features, 0.729）の **約半分**（-54%）。
- ただし評価方法が異なるため直接比較には注意が必要：
  - Group CV は「未知プラグへの汎化」を評価（プラグ間のリークあり）
  - DataRobot TS は「T以前で学習→T以降で評価」の時間的に厳密な評価
- ベストモデルは **Temporal Hierarchical Model with Elastic Net and XGBoost**。
  DataRobot TS 特有のブループリントが選ばれており、時系列構造を活かした学習が
  できていることを示す。
- t+1〜t+7 の 7 点予測の max を閾値判定に使う運用方針において、RMSE 0.337 は
  実用上十分な精度水準と考えられる。
- Group CV の楽観的な RMSE（0.599〜0.729）と比較して DataRobot TS の 0.337 が
  低いのは、時間的リーク除去による精度向上と評価の厳密化の両方が寄与している可能性がある。
